# tetris_lua
Tetris written in Lua using the LÖVE framework

To run the game:

Install LÖVE: Download and install LÖVE from https://love2d.org/.

Run the game:
- Open a terminal/command prompt.
- Navigate to the game directory.
- Run the following command:    love ./
